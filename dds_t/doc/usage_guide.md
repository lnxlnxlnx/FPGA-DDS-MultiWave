# 使用说明

## 1 硬件连接

| 外设 | 接口 | 连接说明 |
|------|------|----------|
| **ADC** (TLC5510) | 8bit 并行 | 模拟信号输入至 ADC 通道 |
| **DAC** (MS9714) | 12bit 并行 | DAC 输出接示波器/后续电路 |
| **时钟** | 12MHz 有源晶振 | PIN_23 输入 |
| **8位数码管** | 共阳 | 显示实时频率 |

### 控制输入

| 按键/开关 | 引脚 | 功能 |
|-----------|------|------|
| `rst` | PIN_51 | 复位，**低有效** |
| `start` | PIN_67 | 启动，下降沿触发 |
| `play` | PIN_53 | 拨码：1=录制，0=回放 |
| `mode` | PIN_54 | 拨码：1=循环播放，0=单次播放 |
| `mode_1` | PIN_59 | 显示切换：0=频率，1=占空比 |

---

## 2 工作模式

### 2.1 直通模式

**条件**: 上电默认；或录制/回放完成后自动回到该状态

**行为**:
- ADC 数据直接映射到 DAC：`da_data[11:4] = ad_data[7:0]`, 低 4 位补零
- ADC/DAC 时钟为 **12MHz 直通**，无采样率限制
- 频率计始终工作，数码管显示实时频率

**用于**: 快速验证 ADC→DAC 通路，或作为实时示波器输出

---

### 2.2 录制模式

**操作步骤**:
1. `play` 拨到 **1** (录制模式)
2. `mode` 拨到 **0** (单次) 或 **1** (循环录制)
3. 按下 `start` 按钮触发录制
4. FPGA 以 **~100kHz** 采样率采集 ADC 数据，存入 4096 深度片内 RAM
5. 单次模式录满自动停止；循环模式持续覆盖录制

**参数**:
- 采样率: ~100kHz (12MHz/120 分频)
- 采样点数: 4096
- 采集窗口: ~41ms
- 录制期间 DAC 保持直通输出

---

### 2.3 回放模式

**操作步骤**:
1. 先完成录制 (或利用上次已存数据)
2. `play` 拨到 **0** (回放模式)
3. 按下 `start` 按钮触发回放

**播放方式** (由 `mode` 拨码选择):

| mode | 模式 | 行为 |
|------|------|------|
| 0 | **单次** | 从 RAM 依次读出 4096 点送到 DAC，播完停止 |
| 1 | **循环** | 播完 4096 点后从头重播，持续循环 |

**回放时钟**: ~100kHz，与录制采样率一致，保证播放速度与原信号同步

---

## 3 频率测量

频率计独立于录制/回放状态，**始终运行**：

- 对 `ad_data` 做迟滞过零检测 (抗噪)
- 1 秒闸门计数 → 频率×10
- 数码管显示格式：**`XXXX.X Hz`** (个位小数点恒亮点亮)

| 显示示例 | 含义 |
|---------|------|
| `0000.0` | 无信号或信号幅值不足 |
| `1000.0` | 1.0000 kHz |
| `0050.0` | 50.0 Hz |

> 频率分辨率 0.1Hz，1 秒刷新一次

---

## 4 典型操作流程

### 场景一：录制一段 41ms 信号并单次回放

```
① 上电复位 (rst 按一下)
② 拨码: play=1, mode=0   (录制模式+单次)
③ 输入模拟信号至 ADC
④ 按 start → 录制开始 (~41ms 自动完成)
⑤ 拨码: play=0           (切换回放模式)
⑥ 按 start → 单次回放 4096 点
```

### 场景二：循环演示

```
① 录制 (同场景 ①~④)
② 拨码: play=0, mode=1   (回放模式+循环)
③ 按 start → 循环回放，直至按 rst 复位
```

### 场景三：占空比测量

```
① 拨码 mode_1=1 (PIN_59 拨到高电平)
② 数码管显示占空比，格式 "XX.X" (如 50.0 = 50.0%)
③ mode_1=0 切回频率显示
```

> 占空比与频率同步测量，每秒刷新一次，通过 mode_1 拨码即时切换显示

---

### 场景四：实时直通+测频

```
① 上电复位 (默认直通)
② 不按 start，拨码任意
③ ADC 输入信号，DAC 直通输出
④ 数码管实时显示频率
```

---

## 5 注意事项

- **复位清零**：`rst` 按下后 RAM 内容不会清零，但 DAC 输出会强制置零；频率计清零重新计数
- **运行时勿切换拨码**：FSM 运行时切换 `play` 或 `mode` 可能导致状态异常，应在空闲时操作
- **未使用的 mode_1**：建议保持悬空或接地
- **DAC 输出范围**：12bit 分辨率的有效值取决于 MS9714 参考电压配置
